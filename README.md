# AU716211_Salling_Pedersen_Mathias
Digital Archives &amp; Methods Exam Project

Project Description:
This repository contains the final project created for the course Digital Archies and Methods in the BA History programme at Aarhus University 2026.

This project examines patterns both through the decades and geographically within the transatlantic slave trade between 1750 and 1800 using quantitative analysis conducted in RStudio. The analysis is based on data obtained from the SlaveVoyages database and focuses on voyage frequency, regions of embarkation and disembarkation as well as mortality rates during voyages.

In addition to the quantitative analysis itself, the project reflects criticially on the strengths and limitations of digital archival datasets and the methodological implications of working with structured historical data.

Repository Structure:

/Data
Contains the original CSV dataset downloaded from SlaveVoyages.

/Scripts
Contains the R script used for:

data cleaning,
mortality rate calculation,
statistical analysis,
and data visualization.

/Output
Contains generated graphs and figures used in the final report.

/Report
Contains the final written assignment and portfolio material.

Open the .Rproj file in RStudio
Run the script located in /Scripts
The visualizations and outputs will be generated automatically
